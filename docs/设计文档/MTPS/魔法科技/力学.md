# 能力分支

---

## 推进力
### 概念
- 推动物体进行运动
### 表现形式
- 控制某个物体，对其施展一个力，推动其运动，可造成(伤害、场景破坏、物体运动)
### 具体可落实点
- 该能力应为术士最基础的能力
- 必须近身接触
- 任意可交互物体均可施加
- 作为常用的伤害与解密来源
